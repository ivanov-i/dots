# Generate Slides

Use the visual-explainer generate-slides prompt template to produce presentation output.

## Workflow

1. Read and follow `../visual-explainer/prompts/generate-slides.md`.
2. Resolve the slide topic and context from user input.
3. Build the deck using the visual-explainer slide workflow and referenced templates.
4. Write output to `~/.agent/diagrams/`, open it in the browser, and report the file path.
